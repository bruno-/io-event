Transfer from the current fiber to the specified fiber. Put the current fiber into the ready list.

Implementation

 
def resume(fiber, *arguments)
	queue = Queue.new(Fiber.current)
	@ready.push(queue)
	
	fiber.transfer(*arguments)
ensure
	queue.nullify
end
